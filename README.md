# Movie Pro
by Roman Duda

Discover the newest and the most popular movies nowadays:

  * Movies' ratings 
  * Detailed movies' descriptions

Technologies implemented:

  * MVVM
  * Retrofit for data requests
  * Data Binding & Parcelable
  * Intents
  * Scroll view
